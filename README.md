## Реализованные возможности в игре
* Передвижение и телепортация игрока
* Разрушение мебели
    + Стулья
    + Столы
    + Бутылки
    + Столовые приборы
    + Прочий интерьер
* Оружия для разрушения
    + Бита
    + Розочка
* Разрезания оленя на части

* Разархивируйте
* Запустите EXE файл
